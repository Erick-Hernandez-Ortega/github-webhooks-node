## Installation

1. Clone this repository
```bash
   git clone https://github.com/Erick-Hernandez-Ortega/github-webhooks-node.git
   cd github-webhooks-node
```

2. Create and configure your `.env` file
```bash
   cp .env.example .env
   # Edit .env with your configuration
```

3. Install dependencies
```bash
   npm install
```

4. Run the application
```bash
   npm start
   # or for development
   npm run dev
```
